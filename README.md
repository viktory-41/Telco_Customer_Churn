# Customer Churn Analysis & Prediction
Анализ оттока клиентов телеком-компании с целью выявления ключевых факторов и разработки рекомендаций по удержанию. 


## Цели
- Выявить факторы, влияющие на отток клиентов
- Построить модель для предсказания риска оттока
- Разработать сегментацию клиентов по уровню риска
- Предложить бизнес-рекомендации

##  Данные
- 7,043 клиента
- 21 признак (демография, услуги, контракт, платежи)
- Источник: Kaggle Telco Customer Churn

##  Ключевые выводы

### Факторы, увеличивающие отток:
1. Помесячный контракт (+40% к риску)
2. Отсутствие техподдержки (+35%)
3. Отсутствие онлайн-безопасности (+30%)
4. Высокие ежемесячные платежи (> $70)

### Факторы, снижающие отток:
1. Двухгодичный контракт (-60%)
2. Наличие техподдержки (-40%)
3. Стаж > 2 лет (-50%)

##  Результаты модели
- **Accuracy**: 81%
- **ROC-AUC**: 85%
- **Точность (precision) для оттока**: 67%
- **Полнота (recall) для оттока**: 55%

## Бизнес-рекомендации

### Для клиентов с помесячным контрактом (32% базы):
Предлагать переход на годовой со скидкой первых 2 месяцев

### Для клиентов без техподдержки (45% базы):
 Бесплатный пробный период на 3 месяца

### Для новых клиентов (< 6 месяцев):
 Усиленная программа онбординга
 Персональный менеджер
